# Installation 
## Nodejs
Version > 10
## UI5 CLI
`npm install --gloabal @ui5/cli`
## Initialize npm project
`npm init -y` -generate package.json
## Initialize UI5 project
`ui5 init` - generates ui5.yaml
## Add manifest file
## OpenUI5/SAPUI5
`ui5 use openui5@latest`
`ui5 use sapui5@latest`
## Run
`ui5 serve` or `npm run start`
